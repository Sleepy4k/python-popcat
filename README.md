# Project Title

Simple request for popcat web, just for fun

## Run Locally  

Clone the project

~~~bash  
  git clone https://github.com/Sleepy4k/python-popcat
~~~

Go to the project directory  

~~~bash  
  cd python-popcat-main
~~~

Install dependencies  

~~~bash  
pip install requests
~~~

Start the server  

~~~bash  
python main.py
~~~
